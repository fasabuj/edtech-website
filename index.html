There are many reasons why you should learn Bootstrap. Bootstrap ensures you have a responsive, mobile-first website. Enabling users to navigate through content on their phones is crucial in anything you build as they generally access websites on their phones more than desktops these days.

Personally, it took me 3–4 days to get a solid grasp on it. When you learn why Bootstrap was created, and what it's designed to do it becomes very easy to learn. Bootstrap has a “Mobile-first” (also known as responsive design) approach when creating websites.

Conclusion. I'd highly recommend that you learn Bootstrap. It is still the most used CSS framework in the world and is incredibly well supported, which is perfect for beginners. If you are going to be building websites, then learning Bootstrap or a similar CSS framework will help you get a job.
